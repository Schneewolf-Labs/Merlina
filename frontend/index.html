🦁 Grimoire Optimizations

                        

                            

                                
                                
                                    Patch model with fused Triton kernels (RMSNorm, RoPE, SwiGLU). Faster training, lower VRAM.
                                
                            

                            

                                
                                
                                    Wrap model with torch.compile for fused kernels (PyTorch 2.x). May add startup overhead.
                                
                            

                        

                        

                            

                                
                                
                                Embedding noise scale for regularization (try 5.0). Leave empty to disable.
                            

                            

                                
                                
                                    Run a baseline eval before training begins.
                                
                            

                        

                        

                            

                                💡 Liger Kernel: Best on Llama, Mistral, Qwen, Gemma, and Phi family models. Requires pip install liger-kernel.
                            

                        

                    


                    
                    

                        
⚡ Attention Implementation

                        

                            
                            
                            
                                Auto mode: Uses Flash Attention 2 on Ampere+ GPUs (RTX 30/40 series, A100, H100), falls back to SDPA or Eager for older GPUs.
                            
                        

                        

                            

                                💡 Performance Guide:

                                • Flash Attention 2: Up to 2-3x faster, lowest memory usage

                                • SDPA: Good balance, works on most modern GPUs

                                • Eager: Slower but most compatible, use for debugging

🔮 GGUF Quantization

                                
                                    After training, Merlina will merge the adapter and brew one GGUF per selected quant. Larger quants keep more quality; smaller quants run on weaker hardware.
                                

                                

                                    
                                    
                                    
                                    
                                    
                                    
                                    
                                    
                                    
                                


                                
                                
                                    The fp16 GGUF is ~2× the size of a 7B model but serves as a high-quality starting point for further quantization. Automatically kept when F16 is checked above.
